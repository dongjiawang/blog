WKWebView是iOS8开始增加的浏览视图

#新特性

性能高，稳定性好，占用的内存比较小 支持JS交互 支持HTML5 新特性 可以添加进度条 支持内建手势 据说高达60fps的刷新频率

#使用 1、导入webkit库

1

#import <WebKit/WebKit.h>

2、遵守协议

WKNavigationDelegate 该代理提供的方法，可以用来追踪加载过程（页面开始加载、加载完成、加载失败）、决定是否执行跳转。

WKUIDelegate 这个代理的方法基本是与界面弹出提示框相关的，针对于web界面的三种提示框（警告框、确认框、输入框）分别对应三种代理方法。 还有就是创建新的webview

7、容易遇到的问题 ➤点击其中的链接没反应

如果一些HTML网页的标签页的跳转方式为

1

<a href = "xxx" target = "_black">

那么点击链接就不会加载 “_black” 的方式是开一个新的页面,就像safari中弹出一个新的页面显示一样 。 解决方法就是重新load，实现下面的代理方式

1
2
3
4
5
6
7

- (void)webView:(WKWebView *)webView decidePolicyForNavigationAction:(WKNavigationAction *)navigationAction decisionHandler:(void (^)(WKNavigationActionPolicy))decisionHandler { 
 if (navigationAction.targetFrame == nil) {
    //重新加载一个新页面
     [webView loadRequest:navigationAction.request]; 
  } 
  decisionHandler(WKNavigationActionPolicyAllow);
}
